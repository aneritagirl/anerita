ANERITA — Code Kit (Master README)

Pieces:
1) app/ — Offline PWA + emergency pages
2) anerita_backend_nextgen/ — Minimal Flask API
3) wallet/ — Apple & Google Wallet templates
4) launchers/ — One‑click scripts to run backend locally

How to demo offline:
- Open app/ANERITA_master_nextgen_PWA.html directly in your browser.
- Or open emergency_tina.html / emergency_gia.html for instant offline pages.

How to run backend locally:
1) Place 'app/', 'anerita_backend_nextgen/', 'wallet/', and 'launchers/' in the same folder.
2) Double‑click the Windows or Mac launcher in 'launchers/'.
3) Once backend starts, set the PWA config to use http://localhost:5000 and click "Sync now".
